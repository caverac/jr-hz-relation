# The structural $J_R$--$h_Z$ slope

Combining the isothermal-sheet scale height $h_Z=\sigma_z^2/(\pi G\Sigma)$ with the
epicyclic action $J_R^{\max}=c_{\max}\sigma_R^2/\kappa$ gives the Palicio et al.
(2024) slope as the squared radial-to-vertical heating-anisotropy ratio,

$$
a = \frac{J_R^{\max}}{h_Z} = c_{\max}\,\frac{\pi G\Sigma}{\kappa}\left(\frac{\sigma_R}{\sigma_z}\right)^2 .
$$

For solar-neighbourhood values ($\Sigma=50\,M_\odot\,\mathrm{pc}^{-2}$,
$\kappa=0.037\,\mathrm{km\,s^{-1}\,pc^{-1}}$, $\sigma_R=35$,
$\sigma_z=18\,\mathrm{km\,s^{-1}}$) and $c_{\max}=1$, this gives
$3.6\times10^{-2}\,L_\odot/\mathrm{kpc}$, within a few per cent of the measured
$3.7\times10^{-2}$.
